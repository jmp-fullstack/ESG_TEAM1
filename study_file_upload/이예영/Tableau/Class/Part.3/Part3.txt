1. 크로스탭

Analytics/ Totals 클릭해서 드래그하면 어느 기준으로 총계를 내줄지 그림으로 알려줌
행, 열, 소개 중복가능

2. Zero Null
마우스 오른쪽누르면서 행열로 드래그하면 Drop Field 생성

빈값을 0으로 만드는법 -> zn(lookup(SUM(수익), 0))
항목이 생성되면 드래그해서 텍스트 위에 올리면 적용가능  

3. 퀵 테이블 계산 및 별도의 범례 사용

컬러값을 주고난 다음에 색상차트 더브클릭하면 Custom Diverging 선택한 후
Stepper Color 를 2단계로만 해주면 대비 가능
또는 한 색으로 통일하면 하나만 가능
